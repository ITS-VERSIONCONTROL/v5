    Rispondere alle seguenti domande:

    1. Cosa fa il comando 'git status'? Spiega come funziona la staging area
    Git status mostra in che branch ci si trova e quanti commit si è a distanza da origin
    La staging area e dove si aggiungo i file pronti al commit.

    2. Descrivi il comando 'git switch <branch>', spiegando brevemente cosa sono i branch e cosa è la HEAD
    Git switch permette di spostare la head da un branch all'altro, la head è il branch corrente in cui si stanno effettuando le modifiche.
    
    3. Cos'è un repository remoto? Descrivi cosa fa il comando 'git clone <remote repository>'
    Un repository remoto è un repository condiviso accessibile da più persone, come ad esempio github.
    il comando git clone permette di creare una copia del repository remoto in locale.
